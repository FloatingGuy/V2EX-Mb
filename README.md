## 添加新功能

1. 本机配置好 [CocoaPods](https://cocoapods.org/)
2. 命令行进入项目文件夹中 `pod install` 下，把第三方库依赖解决掉
3. 打开 V2EX Mb.xcworkspace 
4. 开始玩 (๑´ڡ`๑)
